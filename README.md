# OpenClaw

多频道 AI 网关，支持可扩展的消息集成。

## 快速开始

### 安装

```bash
# npm
npm install -g openclaw@latest

# pnpm
pnpm add -g openclaw@latest
```

### 配置

1. 运行新手引导
```bash
openclaw onboard 
```

2. 登录渠道
```bash
openclaw channels login
```

3. 启动 Gateway
```bash
openclaw gateway 
```
